Gets a transaction from a block hash and transaction hash

#### Request

```ts
{
  blockHash: string;
  transactionHash: string;
}
```

#### Response

```ts
RpcTransaction & {
  noteSize: number
  blockHash: string
  /**
   * @deprecated Please use `notes.length` instead
   */
  notesCount: number
  /**
   * @deprecated Please use `spends.length` instead
   */
  spendsCount: number
  /**
   * @deprecated Please use `notes` instead
   */
  notesEncrypted: string[]
}
```

- [RpcTransaction](/developers/documentation/rpc/objects/rpcTransaction)
