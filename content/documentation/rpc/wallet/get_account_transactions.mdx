Gets transactions for the given account. If not specified, the default account is used.

#### Request

```ts
{
  account?: string
  hash?: string
  sequence?: number
  limit?: number
  offset?: number
  confirmations?: number
}
```

#### Response

```ts
RpcWalletTransaction
```

RPC Objects: 
- [RpcWalletTransaction](/developers/documentation/rpc/objects/rpcWalletTransaction)
