title: wallet/getAccountsStatus
description: RPC Wallet | Iron Fish Documentation
---

Gets the status of an account. If not specified, the status of all accounts are returned.

#### Request

```ts
{
  account?: string
}
```

#### Response

```ts
{
  accounts: Array<{
    name: string
    id: string
    headHash: string
    headInChain?: boolean
    sequence: string | number
  }>
}
```
