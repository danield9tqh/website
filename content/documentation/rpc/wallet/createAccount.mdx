Creates a new account in the wallet with the given name and optionally sets it as the default account.

#### Request

```ts
{
  name: string
  default?: boolean
}
```

#### Response

```ts
{
  name: string;
  publicAddress: string;
  isDefaultAccount: boolean;
}
```
