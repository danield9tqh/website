Gets (and optionally streams) the status of the RPC server

#### Request

```ts
{
  stream?: boolean
} | undefined
```

#### Response

```ts
{
  started: boolean
  adapters: Array<{
    name: string
    inbound: number
    outbound: number
    readableBytes: number
    writableBytes: number
    readBytes: number
    writtenBytes: number
    clients: number
    pending: string[]
  }>
}
```
