Gets (and optionally streams) the status of the long-running jobs queued by the node's workers.
The workers operate similar to a threadpool.

#### Request

```ts
{
  stream?: boolean
} | undefined
```

#### Response

```ts
{
  started: boolean
  workers: number
  queued: number
  capacity: number
  executing: number
  change: number
  speed: number
  jobs: Array<{
    name: string
    complete: number
    execute: number
    queue: number
    error: number
  }>
}
```
